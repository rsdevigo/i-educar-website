&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Para configurar anos escolares, basta ir no **Módulo Escola >> Cadastros Componente Curricular >> Selecionar um componente >> Clicar em Configurar Anos Escolares.**

</div>
